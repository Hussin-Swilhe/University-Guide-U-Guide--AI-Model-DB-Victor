# University-Guide-U-Guide--AI-Model-DB-Victor
It is an intelligent system that relies on artificial intelligence technologies to provide academic and administrative assistance to students at Al-Rasheed Smart University.
هذا المشروع هو مشروع تخرجي انا وبعض زملائي في جامعة الرشيد الذكية – قسم تكنولوجيا المعلومات،

 هو نظام ذكي يعتمد على تقنيات الذكاء الاصطناعي لتقديم المساعدة الأكاديمية للطلاب والإداريين في الجامعات، من خلال واجهة دردشة تفاعلية قادرة على فهم الأسئلة وتقديم إجابات دقيقة وسريعة حول المشاريع، الاختبارات، واللوائح الأكاديمية.
 مساعد ذكي (AI Chatbot): يجيب على الأسئلة الأكاديمية والإدارية بلغة طبيعية وبدقة عالية.

نظام صلاحيات متكامل: واجهات مختلفة للطالب، الإدمن، والسوبر أدمن مع صلاحيات محددة لكل دور.

إدارة المشاريع والاختبارات: إمكانية إضافة المشاريع والاختبارات ومراجعتها وتنظيمها بسهولة.

نظام الإشعارات: يمكن للإدمن إرسال إشعارات عامة أو مخصصة لجميع المستخدمين.

واجهة صوتية تفاعلية: ميزة تحويل الكلام إلى نص والعكس مع واجهة عرض مرئية احترافية.

واجهة حديثة وسلسة: مبنية باستخدام Flutter لتعمل على الويب، الحاسوب، والجوال بكفاءة عالية.

واجهة برمجة تطبيقات آمنة (API): مبنية باستخدام FastAPI مع توثيق JWT لحماية الجلسات.

قاعدة بيانات حديثة: تعتمد على Supabase لإدارة البيانات، الملفات، والاتصال مع نموذج الذكاء الاصطناعي.
